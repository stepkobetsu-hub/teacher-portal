# teacher-portal

個別指導STEP 講師ポータル Ver.1

## ファイル

- index.html
- style.css
- script.js
- images/logo.jpg
- images/nyutaikun.png
- Code.gs（Apps Script用）

## 設定

1. Apps ScriptにCode.gsを貼り付ける
2. ウェブアプリとしてデプロイする
   - 実行ユーザー：自分
   - アクセスできるユーザー：全員
3. 発行されたURLをscript.jsのAPI_URLへ貼る
4. GitHubへindex.html / style.css / script.js / imagesをアップロード
